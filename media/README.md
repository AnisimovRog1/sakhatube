# Легальный тестовый каталог SakhaTube

Эта папка описывает временные ролики для проверки загрузки, HLS-плеера и вертикальной ленты. В неё нельзя добавлять видео, скачанные у авторов с YouTube или других площадок без письменного разрешения/лицензии.

`demo-catalog.json` содержит источник, лицензию и обязательную атрибуцию для каждого ролика. Временные исходники и подготовленные HLS-файлы находятся в `media/source/` и `media/hls/`; они намеренно исключены из Git.

## Локальный конвейер

```bash
npm run media:download-demo
npm run media:package-demo
CONFIRM_DEMO_UPLOAD=yes npm run media:upload-demo
```

Для загрузки в Railway Bucket нужны переменные окружения `S3_ENDPOINT`, `S3_ACCESS_KEY_ID`, `S3_SECRET_ACCESS_KEY`, `S3_BUCKET` и `S3_REGION`.

Пока это тестовый каталог. Перед публичным запуском его нужно удалить из витрины вместе с HLS-файлами либо заменить материалами, на которые SakhaTube владеет правами.
