# SakhaTube iOS

Это нативное приложение для iPhone и iPad: SwiftUI для интерфейса, AVFoundation/AVKit для будущего HLS-плеера, PhotosUI для аватара. Оно не использует WebView.

## Открыть и собрать

1. Откройте `SakhaTube.xcodeproj` в Xcode.
2. Вверху выберите любой iPhone Simulator или свой iPhone.
3. Нажмите кнопку ▶.

Проект сгенерирован из `project.yml`. Если изменится его структура, установите XcodeGen и выполните `xcodegen generate` внутри этой папки.

## Перед TestFlight

- заменить временный bundle ID `com.sakhatube.app` на зарегистрированный в Apple Developer;
- добавить настоящую иконку приложения, скриншоты и ссылку на политику конфиденциальности;
- подключить защищённый вход, публичный каталог, HLS/CDN и серверную проверку покупок;
- заполнить карточку приложения и пройти внутреннее тестирование.
