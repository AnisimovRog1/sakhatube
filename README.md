# SakhaTube

Адаптивный интерфейс первой версии SakhaTube: карусель премьер, каталог сериалов, персональная лента «Для вас», загрузки, профиль и демонстрационный плеер.

## Запуск

Откройте `index.html` в браузере. Проект не требует установки зависимостей.

## Дальше

Интерфейс подготовлен как быстрый фронтенд-слой. Для рабочего сервиса далее подключаются: HLS/CDN, авторизация, каталог, платежи, серверная проверка покупок и веб-админка из продуктового фундамента.

## Studio

Отдельная [демо-админка](admin.html) показывает рабочие сценарии редакции: создание и удаление карточек, порядок главной, очередь файлов, модерацию комментариев и сводные метрики. Изменения сохраняются только в браузере этого устройства. Файлы никуда не отправляются, а вход и роли не являются настоящими — публичную страницу нельзя использовать как боевую админку.

Подробный порядок работ, зависимости и критерии готовности: [SAKHATUBE_DELIVERY_ROADMAP.md](SAKHATUBE_DELIVERY_ROADMAP.md). Контракт для настоящей Studio и backend: [SAKHATUBE_ADMIN_BACKEND_SPEC.md](SAKHATUBE_ADMIN_BACKEND_SPEC.md). Сверенный словарь интерфейса: [SAKHATUBE_LOCALIZATION.md](SAKHATUBE_LOCALIZATION.md).
