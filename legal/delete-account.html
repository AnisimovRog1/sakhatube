Удаление аккаунта и данных

Текущая версия SakhaTube работает без зрительского аккаунта. Если ты оставлял обращение, создавал аккаунт в будущей версии или хочешь удалить данные с нашей стороны, отправь запрос ниже.

Когда аккаунты будут включены, удалить аккаунт можно будет и прямо из настроек приложения. Этот публичный путь остаётся доступен для App Store и Google Play.

Только для локальной разработки

Этот блок существует только для тестирования защищённого цикла и никогда не отображается на опубликованном сервисе.

Запрос никогда не исполняется до подтверждения владельцем e-mail. В текущей тестовой сборке письмо не отправляется: запуск аккаунтов и реальной обработки заявок возможен только после подключения защищённой почтовой доставки. Мы используем e-mail только для ответа по запросу и можем попросить дополнительные сведения, чтобы не удалить данные по чужому обращению. Подробнее — в политике конфиденциальности.
